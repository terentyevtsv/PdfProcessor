КОМАНДЫ ДЛЯ РАЗРАБОТКИ
Пример добавления Миграции:
dotnet ef migrations add InitialCreate --context DocumentsDbContext --output-dir Migrations --startup-project ApiGateway.csproj --project ../Infrastructure/Infrastructure.csproj

Применение миграции:
dotnet ef database update --context DocumentsDbContext --project Infrastructure --startup-project ApiGateway

Установка глобально для ORM:
dotnet tool install --global dotnet-ef

ЗАПУСК ПРИЛОЖЕНИЯ В ОТЛАДЧИКЕ
Нужно установить приложение для работы с Docker-контейнерами, например Docker Desktop.
Поднять окружение командой: "docker-compose up -d", находясь в корне репозитория (директория с файлом docker-compose.yml).

В решении настроить запуск нескольких приложений. Выбрать ApiGateway и BackgroundWorker.
При запуске загрузится стартовая страница swagger.

API
Метод	URL							Описание
POST	/api/documents				Загрузить PDF
GET		/api/documents				Список документов
GET		/api/documents/{id}/text	Получить текст

АРХИТЕКТУРА:
Client -> API Gateway -> RabbitMQ -> BackgroundWorker -> PostgreSQL
